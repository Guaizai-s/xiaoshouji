 核心业务代码

  src/services/claude.js — API 调用，支持 Anthropic 和 OpenAI 两种格式，自动判断环境走不同代理
  src/services/db.js — IndexedDB 数据库，管理角色、会话、消息三类数据
  src/services/speech.js — 语音识别服务（Web Speech API）

  src/views/ChatRoom.vue — 聊天界面（核心页面）
  src/views/Chats.vue — 会话列表
  src/views/Contacts.vue — 通讯录/角色列表
  src/views/NewRole.vue — 新建 AI 角色
  src/views/EditRole.vue — 编辑角色
  src/views/Me.vue — 个人中心
  src/views/Profile.vue — 个人资料编辑
  src/views/Discover.vue / Moments.vue — 发现页 / 朋友圈（UI 框架，暂无实际功能）

  src/components/ — 通用组件：NavBar（顶部栏）、TabBar（底部栏）、ChatInput（输入框）、MessageBubble（消息气泡）

  ---
  配置 & 构建

  vite.config.js — Vite 构建配置，含 PWA 插件设置
  package.json / package-lock.json — 依赖管理
  index.html — 开发入口 HTML
  src/router/index.js — 路由配置（10 个页面路由）
  src/main.js / src/App.vue — Vue 应用入口和根组件

  ---
  部署相关

  api/chat.js — Vercel Serverless Function，生产环境 API 代理（解决 CORS）
  public/_redirects — SPA 路由重定向配置（所有路径指向 index.html）
  proxy-server.js — 本地开发用的代理服务器（localhost:3001）
