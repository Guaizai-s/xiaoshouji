## 🔧 技术细节

### API调用策略
- **开发环境**：使用本地代理服务器（localhost:3001）避免CORS
- **生产环境**：前端直接调用API（纯静态部署）
- **代理服务器**：`proxy-server.js` (Express + CORS)

### 数据库结构（IndexedDB）
```javascript

}

  id, conversationId, role, content, type, timestamp
}
```
