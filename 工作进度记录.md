## 🔧 技术细节

### API调用策略（已更新）
- **开发环境**：使用本地代理服务器（localhost:3001）
- **生产环境**：使用Vercel Serverless Function（/api/chat）
- **自动判断**：根据hostname自动切换

### 关键文件
```
api/chat.js              # Vercel后端函数（解决CORS）
src/services/claude.js   # API调用（自动判断环境）
src/components/MessageBubble.vue  # 气泡组件（表情+分隔）
src/components/NavBar.vue         # 导航栏（纯箭头返回）
src/components/TabBar.vue         # 底部栏（Chrome兼容）
public/emojis/           # 表情图片目录
```

### 数据库结构（IndexedDB）
```javascript
roles: {
  id, name, avatar, apiKey, baseUrl, model, apiFormat, systemPrompt, createdAt, updatedAt
}

conversations: {
  id, roleId, lastMessage, unread, updatedAt
}

messages: {
  id, conversationId, role, content, type, timestamp
}
```
